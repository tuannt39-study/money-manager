# money-manager

Ứng dụng quản lý tài chính cá nhân: cho phép người dùng quản lý chi tiêu, thu nhập, tiết kiệm, đầu tư, tài khoản ngân hàng.

## Tính năng

- Đăng ký
- Đăng nhập
- Quản lý chi tiêu
- Quản lý tài khoản tiền
- Báo cáo
- ...

## Công nghệ

- Jdk 17
- Spring Boot 3
- Spring 6
- Spring Data Jpa 3
- Spring Security 6
- Database H2
- Thymeleaf 3
- HTML, CSS, Javascript
